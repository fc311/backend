This repository contains the **backend** for a Transportation Management App Proof-of-Concept.

It exposes a **GraphQL API** built with **Node.js, Express, GraphQL Yoga**, and **Prisma ORM** using a local **SQLite** database.

## Deployment

- frontend deployment: https://frontend-one-pink-96.vercel.app/
- backend deployment for graphQL: https://transportation-management-app-poc-backend.onrender.com/graphql

## Github Repos

- frontend: https://github.com/fc311/frontend
- backend: https://github.com/fc311/backend

## Hosting

- frontend is deployed on vercel
- backend is deployed with onrender

## Notes

- only admin can see edit actions, employee cannot see edit in the actions in the UI

---

## Tech Stack

- **Runtime:** Node.js
- **API:** GraphQL (GraphQL Yoga)
- **ORM:** Prisma
- **Database:** SQLite
- **Language:** TypeScript

---

## Pre-seeding

The SQLite database file is pre-seeded every time the on-render instance wakes up
